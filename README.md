# bitrix_iblock_translit 


Реализовать обработчик элементов инфоблока с помощью кастомного модуля Битрикс.
1. При установке модуля, он подписывается на событие добавление элемента инфоблока и заполняет поле символьного кода (транслитерацией названия элемента).
Операцию производить только для инфоблока, который указан в настройках модуля (для модуля должны быть своя страница настроек). Считать что элементы инфоблока добавляются либо посредством импорта либо контент-менеджером и требуется изменить (расширить) поведение во врем этого процесса. А именно ввести процесс автоматизации - заполнение символьного кода.
2. Выборку настроек модуля кэшировать.
3. При установке модуль добавляет периодического агента (период запуска - раз в день), который деактивирует элементы инфоблока (п.1) старше определенного периода в днях (задается в настройках модуля).
